# Personal Website

This is a personal website that I created based on an existing design. The goal was to showcase my skills and present my work in a visually appealing way while keeping the site clean and responsive.

## Technologies Used

- **HTML**: Markup language used to structure the content of the website.
- **CSS**: Styling language used to make the website visually appealing.
- **Flexbox**: A CSS layout model used to build flexible and responsive layouts.
- **CSS Grid**: Another layout system that I used for more complex, two-dimensional layouts.
- **SASS**: A preprocessor scripting language that extends CSS, allowing for more dynamic and maintainable stylesheets.

## Development Time

It took me about **1 week** to complete the website. During this time, I worked on building the layout, adding responsive design, and refining the overall aesthetics.
